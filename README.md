# Exam Strategy AI 

An AI-powered web application that analyzes exam question papers to extract:
- Topic-wise distribution
- Overall difficulty score
- Visual insights using charts
- Helps students through AI-powered suggestions

---

##  Live MVP Demo

 **Live URL:**  
https://exam-strategy-ai.onrender.com

 Note: This is an MVP demo link exposed via tunneling for hackathon evaluation.

---

##  Tech Stack

- **Backend:** Flask (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Charts:** charts.js
- **AI Model:** Google Gemini 2.5 Flash
- **Hosting (MVP):** Local Flask + ngrok
- **Version Control:** Git & GitHub
